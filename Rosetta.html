    macro find(key) #

    Finds translations for the given key. The returned object contains translations for every configured locale:

    Rosetta.find("user.name")
# => { "en" => "Name", "es" => "Nombre", "nl" => "Naam" }

    If the key does not exist, a compile error will be raised.
